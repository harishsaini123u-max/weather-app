# 🌦 Weather App

A simple weather app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Search any city
- Temperature display
- Humidity and wind speed
- Dynamic weather icons

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API
